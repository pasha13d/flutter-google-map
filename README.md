# google_map

Functionality:
- map
- locations searching

Tools / packages:
- Flutter
- flutter_map
- positionstack.com
- BLoC
- Very good CLI
- Autocomplete widget
- l10n (translations)
- Serialization (json_annotation)
- http
- REST API

Description:
App supports standards of writing code. This project is divided into 3 layers:
- Data Layer (Location API CLient)
- Domain Layer (Locations Repository)
- Presentations Layer (widgets + BLoC)
